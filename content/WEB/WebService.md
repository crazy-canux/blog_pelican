# webservice

webservice用于server和browser之间通讯．

webservice是基于http的，无状态的．

webservice一般分为soap webservice和http webservice.

***

# SOAP

Simple Object Access Protocol, 简单对象访问协议，是一个基于XML的协议．

简单来说SOAP的思想就是通过http + XML来在server和browser之间交换信息．
